# Paintings Folder

This folder contains images for the paintings section of the portfolio.

## Image Naming Convention

Please place your painting images in this folder with the following naming convention:

- `sunset-dreams.jpg`
- `urban-rhythm.jpg`
- `natures-whisper.jpg`
- `geometric-harmony.jpg`
- `emotional-landscape.jpg`
- `silent-contemplation.jpg`

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Image Recommendations

- Use high-quality images (minimum 1200px width)
- Maintain consistent aspect ratios
- Optimize file sizes for web performance
- Use descriptive alt text in the service file
